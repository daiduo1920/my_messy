pairs正交：

1\. 制作取值表（只列出数据即可，不用编号）

2\. 复制取值表的数据，放到文本文档中保存（注意不要更改任何格式，例如文件叫test.txt ）

3\. 把文本文档放在allpairs文件夹中

4\. win+r后输入cmd进入控制台

5\. 使用控制台代码进入allpairs文件夹（cd 目录名字）

6\. 在控制台中输入allpairs.exe test.txt>chenggong.txt （ chenggong是自己起的名字，用来存

放生成的组合用例，可以自动生成，不必提前建好）



格式：



体型	年龄段	性别

胖	老年	男

适中	青年	女

瘦	儿童
